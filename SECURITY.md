# Security Policy

## Reporting a Vulnerability

**Please do NOT report security vulnerabilities through public GitHub issues.**

To report a security vulnerability:

1. Email [hello@pushfire.app](mailto:hello@pushfire.app) with subject: `[SECURITY] PushFire Firebase Extension`
2. Include details about the vulnerability
3. We'll acknowledge receipt within 48 hours

## Security Updates

We'll notify users of security updates through:

- GitHub Security Advisories
- Email notifications

---

For security concerns, contact [hello@pushfire.app](mailto:hello@pushfire.app)
